# ことばかけ

高齢者向け・孤独感の軽減とフレイル予防を目的としたLINE bot。

「やさしい診断」とは別プロダクト。WEBサイト診断ではなく、高齢者本人とAIの日常的な対話を通じて、
孤独感の軽減とフレイル予防（低栄養・活動量低下・社会的孤立の早期発見）を目指すサービス。

## コンセプト

- 高齢者は能動的にチャットボットを使わない前提で設計する
- AI側から話しかける「声かけ」を基本動作にする
  - 一定時間（例: 24時間）投稿がなければこちらから確認の声かけをする
  - 朝の決まった時間（例: 10時）までにやり取りがなければ挨拶を送る
- センサーやウェアラブルによる物理的な安否確認は行わない（スコープ外）
  - 最終会話時刻をDBに記録し、経過時間に応じてプッシュ通知・声かけを行う仕組みのみで実現する
- 会話内容そのものではなく「今日も会話があった」という事実レベルの情報のみを家族に共有する設計を想定（本人のプライバシーとのバランス）
- 指示・助言をしない、聞き役に徹するトーンを意識する

## 技術スタック（想定）

- LINE Messaging API（Webhook / プッシュメッセージ）
- Supabase（DB・Edge Function、他プロダクトと共通のプロジェクトを利用予定）
- 対話生成はAnthropic API等を想定
